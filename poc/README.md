# IGNIS temporary proof of concept

This is the active scope until the camera/model/stream path is proven on the QNX Raspberry Pi.

```text
Pi camera -> local TFLite detector -> fire/smoke boxes -> authenticated TCP -> laptop dashboard
```

The POC deliberately excludes voice, phone calls, alarms, zone editing, evidence packaging, and the production multi-process supervisor. Those files remain in the repository but are frozen. Start with [POC_HANDOFF.md](../docs/POC_HANDOFF.md).
