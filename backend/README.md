# IGNIS backend

FastAPI owns laptop persistence, browser broadcast, typed response actions, voice access, and demo-call safety. It does not decide whether the camera observed a sustained hazard.

```powershell
python -m venv .venv
.venv\Scripts\python -m pip install -r requirements-dev.txt
Copy-Item .env.example .env
.venv\Scripts\python -m uvicorn app.main:app --reload
```

HTTP listens on 8000 and the authenticated device protocol on 9001. Calls remain disabled unless every server-side safety condition is configured.
